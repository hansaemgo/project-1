## JavaScript 입문

#### 01. 변수

- 특정 이름에 특정 값을 담을 때 사용
- 변수는, 바뀔수 있는 값을 말하며, 한번 선언하고 나서 바꿀 수 있음

  ```javascript
  console.log('hello'); // hello
  let value = 1;
  console.log(value); // 1
  value = 2;
  console.log(value); // 2
  ```

- 선언시 `let`이라는 키워드 사용 --한번선언 시 같은이름 선언 불가--

  ```javascript
  let value = 1;
  let value = 2;
  ```
